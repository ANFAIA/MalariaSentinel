# LinkedIn Post Generator

Generates a LinkedIn post to accompany a visual presentation (carousel, video, slides) of a project.

## Input

A text describing the project — what it is, what it does, current status, funding, tech stack, anything relevant. Raw notes are fine.

## Output

A LinkedIn post in two versions (English + Spanish) following this structure:

1. **Hook** — casual, first-person opening. "Working on something I'm excited about" energy.
2. **What it is** — one or two sentences explaining the project in plain language. No jargon dumps. If technical terms are necessary, briefly contextualise them.
3. **Honest status** — where it is today, what's working, what's not. Authenticity over hype.
4. **Acknowledgements** — funders, collaborators, tools. Keep it brief.
5. **Call to action** — open DMs, feedback welcome, collaboration invite. Low pressure.
6. **Link** — GitHub or relevant URL.
7. **Hashtags** — 3 to 5 relevant ones.

## Tone rules

- Conversational but not casual-professional LinkedIn slop. No "thrilled to announce", no "game-changer", no "leveraging".
- First person. Personal. The reader should feel like someone is talking to them, not at them.
- Honest about limitations. Weaknesses are features of credibility, not things to hide.
- Technical details are welcome when they serve the story, not when they're a flex.
- No emojis in the body. Hashtags only.
- Keep it under 150 words per language version.

## Format

Return both versions separated by a horizontal rule. The Spanish version is a translation, not a rewrite — same structure, same energy.

```
[English version]

---

[Versión en español]
```
