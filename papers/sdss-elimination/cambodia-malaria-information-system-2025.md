# A bespoke, real-time surveillance system for malaria elimination in Cambodia

**Authors:** Cambodia National Malaria Control Program
**Journal:** PLOS ONE | **Year:** 2025 | **DOI:** PMC12888552
**File:** papers/sdss-elimination/cambodia-malaria-information-system-2025.md

---

## Abstract

This paper describes a locally built digital platform integrating surveillance, diagnostics, treatment, logistics, entomology, and operational monitoring for malaria elimination in Cambodia. Mobile applications enable real-time reporting even in offline conditions, supporting the 1-3-7 surveillance model (case notification within 1 day, investigation within 3 days, response within 7 days) and decentralised decision-making.

## Methods

- Bespoke digital platform development with modular architecture
- Mobile applications for field-level data collection (offline-capable)
- Integration of six functional modules: surveillance, diagnostics, treatment, logistics, entomology, operational monitoring
- 1-3-7 surveillance model implementation
- Decentralised decision-making framework with national oversight
- Real-time data synchronisation and analytics

## Key Results

- Successful deployment of real-time surveillance system across Cambodia's malaria programme
- Offline mobile applications enabling field reporting in remote areas
- 1-3-7 model operationalised through digital workflow management
- Integrated entomological and clinical data for combined analysis
- Decentralised decision-making supported by real-time dashboards

## Relevance to MalariaSentinel (Centinela)

Modern real-time surveillance system demonstrating digital health transformation for elimination. The modular architecture (surveillance, diagnostics, treatment, logistics, entomology, operations) provides a blueprint for the Centinela's module design. The 1-3-7 model implementation demonstrates how digital systems can operationalise surveillance targets. The offline-capable mobile approach is relevant for Ghana's rural settings with limited connectivity. The integration of entomological and clinical data aligns with the Centinela's multi-data-source approach.

## Limitations

- Cambodia-specific development; may not be directly transferable
- Bespoke development requires sustained local technical capacity
- Limited description of predictive analytics or modelling capabilities
- Scalability to larger populations not fully demonstrated

## Future Directions

- Add ABM-based predictive simulation capabilities
- Extend to cross-border surveillance coordination
- Integrate AI/ML for automated case classification and risk prediction
- Develop open-source version for broader adoption
